## 🔧 本地开发

### 确保环境

- **Node.js** - [`v18.0.0`](https://nodejs.org/en) 或更新版本
- **文本编辑器** - 推荐使用 [Visual Studio Code](https://code.visualstudio.com/) 以及 [Volar 扩展](https://marketplace.visualstudio.com/items?itemName=Vue.volar)

确保安装依赖项：

```shell
pnpm install
```

启动开发服务

```shell
pnpm run dev
```
